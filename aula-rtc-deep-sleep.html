    Como o ESP8266 Dorme

    3 modos de sleep — do cochilo ao desligamento

    Os 3 Modos de Sleep

    Modo Consumo CPU Wi-Fi Acordar com
    Modem Sleep ~15 mA Ligada Off Timer automático
    Light Sleep ~0,4 mA Pausada Off Timer ou GPIO
    Deep Sleep ~20 uA Off Off GPIO 16 (RST)

    Deep Sleep desliga tudo. Só o RTC continua. Maior economia, mas reinicia do zero.

    Deep Sleep em Detalhe

  • Tudo é desligado (CPU, Wi-Fi, RAM)
  • Só o RTC continua rodando
  • Consome apenas 20 uA (0,02 mA)
  • Acorda como se fosse ligado do zero
  • Roda setup() novamente
    •  
    graph LR
  A["setup()"] -->|"Trabalha"| B["Lê/Envia"]
  B -->|"ESP.deepSleep()"| C["Dorme 20 uA"]
  C -->|"RST pulso"| A

    O Pino Mágico: GPIO 16 para RST

  • Conexão obrigatória: D0 (GPIO 16) no RST
  • É o RTC que "cutuca" o RST para acordar
  • Sem esse fio: o ESP dorme para sempre
  • Resistor 10k entre RST e 3.3V (pull-up)
    •

    Calculadora de Autonomia

    Dados:

  • Bateria 18650: 3000 mAh
  • Acordado (Wi-Fi): 80 mA por 5 seg
  • Dormindo: 0,02 mA
  • Ciclo: a cada 30 minutos
    •

    Calculo por dia:

  • 48 ciclos/dia (24h / 30min)
  • Acordado: 80 x 4min/60 = 5,3 mAh
  • Dormindo: 0,02 x 23,93h = 0,48 mAh
  • Total: ~5,8 mAh/dia
    •

    Autonomia: 3000 / 5,8 = 517 dias

    Como lembrar que horas sao sem Wi-Fi?

    RTC Memory — O "Post-it" do ESP8266

  • 512 bytes que sobrevivem ao deep sleep
  • Declarado com RTC_DATA_ATTR
  • Rapido como RAM (sem desgaste de flash)
  • Nao sobrevive a desconexao de energia
    •  
    
RTC_DATA_ATTR int contador = 0;
RTC_DATA_ATTR float ultimaTemperatura = 0;
RTC_DATA_ATTR bool primeiraVez = true;

    NTP Inteligente — A Economia

  • 1a vez: conecta Wi-Fi, sincroniza NTP, salva epoch
  • Depois: calcula hora via millis() - millisBase + epochBase
  • Economia: Wi-Fi ligado so no primeiro ciclo
    •

    Com Wi-Fi: 5 seg x 80mA = 400 mA.s. Sem Wi-Fi: 5ms x 20mA = 0,1 mA.s = 4.000x menos por ciclo

    Codigo: NTP + RTC Memory

    
RTC_DATA_ATTR bool ntpOk = false;
RTC_DATA_ATTR time_t epochBase = 0;
RTC_DATA_ATTR unsigned long millisBase = 0;

void setup() {
  Serial.begin(115200);

  if (!ntpOk) {
    // Primeira vez: conecta e sincroniza
    WiFi.begin(ssid, senha);
    configTime(0, 0, "pool.ntp.org");
    while (time(nullptr) < 1000) delay(100);
    epochBase = time(nullptr);
    millisBase = millis();
    ntpOk = true;
    WiFi.disconnect(true);  // Desliga Wi-Fi!
  }

  // Calcula hora sem Wi-Fi
  time_t agora = epochBase + (millis() - millisBase) / 1000;
  Serial.printf("Hora: %s", ctime(&agora));

  ESP.deepSleep(300e6);  // 5 minutos
}

void loop() {}

    Codigo NTP — Linhas-Chave

    
RTC_DATA_ATTR bool ntpOk = false;
RTC_DATA_ATTR time_t epochBase = 0;
RTC_DATA_ATTR unsigned long millisBase = 0;

void setup() {
  Serial.begin(115200);

  if (!ntpOk) {
    WiFi.begin(ssid, senha);
    configTime(0, 0, "pool.ntp.org");       // sincroniza
    while (time(nullptr) < 1000) delay(100); // espera NTP
    epochBase = time(nullptr);
    millisBase = millis();
    ntpOk = true;
    WiFi.disconnect(true);
  }

  time_t agora = epochBase +                // calcula sem Wi-Fi
    (millis() - millisBase) / 1000;          // diferenca de tempo
  Serial.printf("Hora: %s", ctime(&agora));
  ESP.deepSleep(300e6);
}
