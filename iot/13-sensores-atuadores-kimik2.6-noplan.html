    Sensores e atuadores que já usamos em aulas passadas

    LDR + DHT22 🌡️

    • LDR: resistor dependente de luz (analógico)
    • DHT22: temperatura e umidade (digital)
    • Leitura analógica: analogRead(pin)
    • DHT22 usa biblioteca DHT.h

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    LCD I2C + Relé 🔌

    • LCD I2C: display 16x2 ou 20x4 via I2C
    • Endereço comum: 0x27 ou 0x3F
    • Relé: chave elétrica controlada por GPIO
    • Controla AC: lâmpadas, eletrodomésticos

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    Motor DC + MOSFET ⚙️

    • Motor DC: rotação contínua, PWM controla velocidade
    • MOSFET: transistor para chaveamento de alta corrente
    • L298N: driver para controle de direção e velocidade
    • Velocidade: analogWrite(pin, dutyCycle)

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    Sensores Simples

    Alternativas para medições básicas

    DS18B20 🌡️

    • Sensor digital de temperatura (OneWire)
    • Faixa: -55°C a +125°C
    • Precisão: ±0.5°C (de -10 a 85°C)
    • Versão à prova d'água disponível
    • Múltiplos sensores no mesmo pino!

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    DS18B20 — Código Mínimo

    Boia e Sensores de Nível 🌊

    • Sensor de boia: switch magnético
    • Detecta nível máximo/mínimo de líquido
    • Saída digital (HIGH/LOW)
    • Usos: caixa d'água, aquário, reservatório
    • Similar: sensor de chuva, sensor de umidade do solo

    Sensores de Distância

    Medir proximidade, nível de enchimento, obstáculos

    HC-SR04 Ultrassônico 📏

    • Princípio: eco de onda ultrassônica (40 kHz)
    • Alcance: 2 cm a 4 m
    • Precisão: ±0.3 cm
    • Pinos: VCC, Trig, Echo, GND
    • Preço: ~R$ 5-15

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    Sensores Ambientais

    Monitorar o ambiente ao redor do dispositivo

    MQ-2 — Gás e Fumaça 🔥

    • Detecta: fumaça, H2, álcool, metano, GLP
    • Saída analógica (A0) + digital (D0)
    • Precisa de aquecimento (~30s)
    • Usos: detecção de vazamento, incêndio
    • Preço: ~R$ 10-20

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    Umidade do Solo + Chuva 🌱

    • Sensor de umidade do solo
    • Mede condutividade entre duas sondas
    • Saída analógica (mais úmido = menor valor)
    • Sensor de chuva: placa com detecção de gotas
    • Usos: irrigação inteligente, alerta de chuva

    ⚡ Dica importante

    Para evitar corrosão, energize o sensor de solo só durante a leitura (GPIO como fonte de alimentação)

    MPU6050 — Acelerômetro + Giroscópio 📐

    • 3 eixos de aceleração + 3 eixos de giroscópio
    • Comunicação I2C (endereço 0x68 ou 0x69)
    • Temperatura interna inclusa
    • Usos: detecção de queda, inclinação, vibração
    • Preço: ~R$ 10-20

    🎮 Aplicações criativas

    Controle de gestos, braço robótico, estabilização de drones, smartwatch

    Sensor Hall + PIR 🧲👤

    Sensor Hall

    • Detecta campo magnético
    • Usos: contagem de rotações, portas, proximidade
    • ESP32 tem sensor Hall interno!

    Sensor PIR

    • Detecta movimento por infravermelho
    • Saída digital (HIGH ao detectar)
    • Ajuste de sensibilidade e tempo
    • Usos: alarmes, acendimento automático

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    Atuadores — Motores e Movimento

    Fazer o mundo físico reagir aos dados dos sensores

    Servo Motor SG90 🎯

    • Posicionamento preciso de 0° a 180°
    • Controlado por PWM (50 Hz)
    • Pequeno, barato (~R$ 10), fácil de usar
    • Usos: abertura de portas, robótica, câmera pan/tilt
    • Biblioteca: ESP32Servo.h

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    Servo — Código Mínimo

    Motor de Passo 28BYJ-48 🔄

    • Rotação em passos discretos (precisão angular)
    • 2048 passos por volta (0.18°/passo)
    • Driver ULN2003 incluso
    • Usos: impressora 3D, CNC, robótica, relógios
    • Preço: ~R$ 15-25 (com driver)

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    Solenóide e Comparativo de Atuadores 🔩

    Solenóide

    • Eletroímã linear: empurra/puxa uma haste
    • Movimento curto e rápido
    • Usos: fechaduras, válvulas, dispensers
    • Requer driver (corrente alta)

    Quando usar?

    • Servo: posicionamento angular preciso
    • Stepper: rotação contínua precisa
    • DC + MOSFET: velocidade variável
    • Solenóide: movimento linear rápido
    • Relé: ligar/desligar AC

    Comunicação IoT

    Conectando o projeto além do Wi-Fi

    LoRa 📡

    • Long Range: até 15 km (campo aberto)
    • Baixo consumo de energia
    • Baixa largura de banda (pequenos pacotes)
    • Usos: agricultura, monitoramento remoto, smart cities
    • TTGO LoRa32: ESP32 + LoRa + OLED integrados

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    Infravermelho e Comparativo 📶

    IR (Infravermelho)

    • Controle remoto de TVs, aparelhos
    • Comunicação curta distância (5-10 m)
    • Receptor IR + controle remoto
    • Biblioteca: IRremoteESP8266

    Comparativo

    • Wi-Fi: curto alcance, alta taxa
    • LoRa: longo alcance, baixa taxa
    • Bluetooth: curto alcance, baixo consumo
    • IR: linha de vista, barato
    • MQTT: protocolo de mensagens (sobre TCP/IP)

    Exemplo de Justificativa 📝

    Cenário: Estação meteorológica rural (sem energia)

    • Temperatura: DS18B20 (preciso, à prova d'água, 1-Wire)
    • Umidade do solo: sensor capacitivo (mais durável que resistivo)
    • Atuador: relé para bomba d'água (controle AC)
    • Comunicação: LoRa (longo alcance, baixo consumo)
