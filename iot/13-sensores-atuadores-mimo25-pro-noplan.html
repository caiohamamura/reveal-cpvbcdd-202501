    Revisão: LDR + DHT22

    LDR (Luminosidade)

    • Leitura analógica (0-1023)
    • Quanto mais luz, menor a resistência

    DHT22 (Temp + Umidade)

    • Leitura digital (protocolo próprio)
    • Precisão: ±0.5°C, ±2% UR

    Revisão: LCD I2C + Relé

    LCD I2C (Display)

    • Comunicação I2C (2 fios: SDA, SCL)
    • Endereço padrão: 0x27 ou 0x3F

    Relé (Chaveamento)

    • Controla cargas AC/DC de alta potência
    • Sinal digital: HIGH = liga, LOW = desliga

    Revisão: Motor DC + MOSFET

    Motor DC

    • Controle de velocidade via PWM
    • Controle de direção via Ponte H

    MOSFET (IRF520)

    • Atua como chave eletrônica
    • Suporta correntes altas (>1A)
    • Pino PWM controla rotação

    Etapa 2: Sensores Simples

    Evoluindo do familiar — novos sensores básicos

    DS18B20 — Temperatura Impermeável

    • Protocolo 1-Wire (1 pino de dados)
    • Faixa: -55°C a +125°C
    • Precisão: ±0.5°C
    • Versão à prova d'água (sonda inox)
    • Vários no mesmo pino (endereçamento)

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    DS18B20 — Código

    Bibliotecas: OneWire + DallasTemperature

    Sensor de Boia — Nível de Água

    • Sensor magnético simples (contato seco)
    • Digital: HIGH ou LOW
    • Boia sobe com nível → fecha/abre contato
    • Ideal para: caixa d'água, reservatórios
    • Sem consumo de energia (passivo)

    HC-SR04 — Ultrassônico

    • Funciona com ecos de som (40kHz)
    • Faixa: 2cm a 400cm
    • Precisão: ±3mm
    • Pinos: Trig + Echo
    • Projetos: robô seguidor, estacionamento, medição

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    HC-SR04 — Código

    Sensor IR — Infravermelho de Obstáculo

    • Emissor + receptor IR integrados
    • Digital: detecta presença/ausência
    • Faixa: 2cm a 30cm (ajustável)
    • Usos: seguidor de linha, obstáculo, contagem
    • ⚠️ Sensível à luz solar direta

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    VL53L0X — Sensor a Laser (ToF)

    • Time of Flight (ToF) — mede tempo da luz
    • Faixa: 3cm a 200cm
    • Precisão: ±3% (muito superior ao IR)
    • Comunicação: I2C
    • Imune à luz ambiente
    • Projetos: drone, robótica, gesture control

    Fonte: Adafruit Learning System (CC BY-NC-SA 3.0)

    RCWL-0516 — Radar Doppler (Micro-ondas)

    • Detecta movimento por efeito Doppler
    • Alcance: ~7 metros
    • Atravessa paredes finas e plástico
    • Digital: HIGH quando detecta movimento
    • Alimentação: 4-28V DC
    • Complemento ideal ao PIR

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    Etapa 4: Sensores Ambientais

    Monitorando o meio ambiente — gás, solo, chuva, movimento e mais

    MQ-2 — Gás e Fumaça

    • Detecta: GLP, CO, hidrogênio, fumaça
    • Saídas: analógica (intensidade) + digital (limiar)
    • Necessita pré-aquecimento (~20s)
    • Potenciômetro ajusta sensibilidade
    • Projetos: alarme de incêndio, detecção de gás

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    Sensor de Umidade do Solo

    Capacitivo (recomendado)

    • Não sofre corrosão
    • Saída analógica proporcional
    • Alimentação: 3.3-5V

    Resistivo (mais barato)

    • Corroi com o tempo
    • Menos preciso

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    Sensor de Chuva (FC-37)

    • Placa com trilhas paralelas
    • Água = conduz corrente = variação analógica
    • Quanto mais chuva, menor a leitura
    • Digital: módulo comparador com LM393
    • Projetos: irrigação automática, alerta enchente

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    MPU6050 — Acelerômetro + Giroscópio

    • 6 eixos: 3 aceleração + 3 rotação
    • Comunicação: I2C (0x68)
    • Detecta: inclinação, vibração, movimento
    • Sensor de temperatura integrado
    • Projetos: drone, controle de gestos, pedômetro

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    Sensor Hall + PIR (Movimento)

    Sensor Hall (Efeito Hall)

    • Detecta campo magnético
    • ESP32 tem Hall integrado!
    • Usos: portas, RPM, sensores de posição

    HC-SR501 (PIR)

    • Detecta calor corporal (infravermelho)
    • Alcance: ~7m, ângulo: 120°
    • Ajuste de sensibilidade e tempo

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    Servo Motor SG90

    • Controle de posição angular (0°-180°)
    • Sinal PWM (50Hz, 1-2ms pulso)
    • Torque: 1.8 kgf·cm (SG90)
    • Alimentação: 5V (pelo Arduino OK)
    • Projetos: braço robótico, portão, câmera PTZ

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    Motor de Passo 28BYJ-48 + ULN2003

    • Precisão angular por passo (5.625°/64)
    • Modo half-step: 4096 passos/revolução
    • Driver ULN2003 integrado
    • Torque baixo, mas muito preciso
    • Projetos: impressora 3D, câmera giratória, esteira

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    Solenóide — Movimento Linear

    • Converte energia elétrica em movimento linear
    • Tipos: push (empurra) e pull (puxa)
    • Controle via relé ou MOSFET
    • Corrente alta (500mA-2A) — precisa driver
    • Projetos: fechadura elétrica, dispensador, válvula

    Etapa 6: Comunicação IoT

    Conectando dispositivos — LoRa e Infravermelho

    LoRa — Long Range Communication

    • Alcance: até 15km (campo aberto)
    • Baixo consumo de energia
    • Frequência: 868/915 MHz (varia por região)
    • Taxa baixa: 0.3 a 50 kbps
    • Protocolo SPI (com ESP32)
    • Projetos: agricultura, cidades inteligentes, rastreamento

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

    IR — Controle Remoto (Receptor)

    • Receptor IR: VS1838B / TSOP4838
    • Decodifica sinais de controles remotos
    • Protocolos: NEC, Sony, RC5, Samsung...
    • 3 pinos: VCC, GND, Signal
    • Projetos: controle de LEDs, TV smart, robô com controle

    Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)
