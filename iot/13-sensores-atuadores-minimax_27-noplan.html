Revisão: O que Já Conhecemos

Base para a aula de hoje

Revisão: LDR + DHT22

  • LDR — Resistor fotocondutivo (dependente de luz)
  • Leitura analógica via analogRead()
  • DHT22 — Temperatura + Umidade
  • Comunicação digital via DHT.read22()
  • Precisão: ±0.5°C, ±2% umidade

Imagem: RandomNerd Tutorials

Revisão: LCD I2C + Relé

  • LCD I2C — Display 16x2 via barramento I2C
  • Poucos pinos: SDA + SCL
  • liquidCrystal_I2C.write()
  • Relé — Controle de corrente alta
  • digitalWrite(pino, HIGH/LOW)
  • Isolamento galvânico entre MCU e carga

Imagem: RandomNerd Tutorials

Revisão: Motor DC + MOSFET

  • Motor DC — Rotação contínua
  • Controle de velocidade com analogWrite(PWM)
  • MOSFET — Chave eletrônica de alta potência
  • IRL540N ou similar para Arduino
  • Gate = PWM do Arduino
  • Drain = Motor, Source = GND

Fonte: Random Nerd Tutorials

Sensores Simples

Evoluindo do familiar

DS18B20 — Sensor de Temperatura 1-Wire

  • Comunicação 1-Wire (1 pino!)
  • Vários sensores em cascata
  • Precisão: ±0.5°C (-10°C a 85°C)
  • Faixa: -55°C a 125°C
  • Biblioteca: DallasTemperature

Imagem: RandomNerd Tutorials

Boia — Nível de Água

  • Sensor binário: ATIVADO / DESATIVADO
  • Baseia-se em float switch (boia magnética)
  • Sem contato com a água diretamente
  • Ideal para reservatórios, tanks
  • Saída: digitalRead()
  • Baixo custo: R$ 5 a R$ 15

Imagem: Particle Community

Sensores de Distância

Medindo o mundo ao redor

HC-SR04 — Sensor Ultrassônico

  • Mede distância por tempo de voo do som
  • Faixa: 2cm a 400cm
  • Precisão: ±3mm
  • 4 pinos: VCC, GND, TRIG, ECHO

Imagem: RandomNerd Tutorials

Sensores IR — Infravermelho

  • Sharp GP2Y0A21YK — analog output
  • Faixa: 10cm a 80cm
  • Saída analógica: tensão proporcional à distância
  • Mede distância por reflexão
  • Boa precisão em curtas distâncias
  • Sensível a luz ambiente e cor das superfícies

Imagem: Adafruit Learning System

VL53L0X — Sensor Laser ToF

  • Time of Flight — luz, não som!
  • Faixa: 50mm a 1000mm (precisão ±5%)
  • Interface: I2C (SDA, SCL)
  • Campo de visão: 25°
  • Precisão superior ao ultrassônico
  • Ideal para exteriores (luz solar)

Imagem: Adafruit Learning System

RCW-0001 — Sensor Doppler (Radar)

  • Detecta movimento por micro-ondas
  • Não precisa contato direto
  • Detecta através de madeira, plástico, vidro
  • Faixa: até 9 metros
  • Alimentação: 5V, consumo ~30mA
  • Útil para alarmes, automação

Imagem: Banggood

Sensores Ambientais

Monitorando o ambiente

MQ-2 — Sensor de Gás e Fumaça

  • Detecta: GLP, butano, propano, metano, fumaça
  • Saída analógica + digital (threshold)
  • Sensibilidade ajustável via potenciômetro
  • Aquecimento necessário (~5 min)
  • Cuidado: não é à prova de explosão!

Imagem: RandomNerd Tutorials

Sensor de Umidade do Solo

  • Mede resistência elétrica do solo
  • Solo seco = alta resistência
  • Solo úmido = baixa resistência
  • Saída: analógica (0-1023)
  • Construção simples: dois pinos no solo
  • Ideal para irrigações automáticas

Imagem: RandomNerd Tutorials

Sensor de Chuva

  • Placa com trilhas condutivas
  • Gotas de chuva = curto circuito
  • Saída analógica: mais chuva = menor valor
  • Saída digital: threshold configurável
  • Módulo com LM393 comparator
  • Baixo custo: R$ 5 a R$ 10

Imagem: RandomNerd Tutorials

MPU-6050 — Acelerômetro + Giroscópio

  • 6 eixos: 3 aceleração + 3 rotação
  • Interface: I2C
  • Comunicação: protocolo I2C robusto
  • Aplicações: gestos, orientação, robôs
  • Biblioteca: Adafruit_MPU6050

Imagem: Adafruit Learning System

Sensor Hall — Detecção de Campo Magnético

  • Detecta campo magnético permanente
  • Sensor linear ou digital (on/off)
  • Sem contato físico — vida útil longa
  • Aplicações: encoders, contagem de rotações
  • Usado em motores brushless
  • Exemplo: A3144 (digital Hall effect)

Imagem: Hobby components

PIR — Sensor de Presença (Infravermelho Passivo)

  • Detecta pessoas/animais por calor corporal
  • PIR = Passive Infrared
  • Lentes Fresnel concentram IR
  • Alcance: 3 a 7 metros
  • Saída digital: HIGH quando detecta
  • Ideal para: alarmes, luzes automáticas

Imagem: RandomNerd Tutorials

Atuadores: Motores e Movimento

Transformando sinais em movimento

Servo Motor

  • Posicionamento angular preciso
  • 0° a 180° (ou rotação contínua)
  • Controle via PWM (25ms cycle)
  • 3 fios: VCC, GND, Signal
  • Biblioteca: Servo.h
  • Usos: robôs, válvulas, direção

Imagem: RandomNerd Tutorials

Motor de Passo

  • Movimento em passos discretos
  • Alta precisão angular (1.8° por passo)
  • Driver necessário: A4988 ou DRV8825
  • 2 fases: bipolar ou unipolar
  • Torque alto em baixas rotações
  • Usos: CNC, impressoras 3D, robôs

Imagem: RandomNerd Tutorials

Solenóide — Força Linear

  • Conversão eletromagnética → força linear
  • Atuação bistável ou monostável
  • Força proporcional à corrente
  • Usos: travas, disjuntores,bombas
  • ATENÇÃO: consumo alto de corrente
  • Necessita driver (transistor/MOSFET)

Imagem: RandomNerd Tutorials

Motor DC com Ponte H (Direção)

  • Ponte H — controla direção e velocidade
  • L298N ou L293D drivers comuns
  • 2 pinos de direção + 1 PWM
  • Giro horário ou anti-horário
  • Freio elétrico (curto-circuito)
  • Usos: carros RC, esteiras, robôs

Imagem: RandomNerd Tutorials

Comunicação IoT

Conectando o mundo

LoRa — Long Range Radio

  • Comunicação longa distância (km!)
  • Baixo consumo de energia
  • Sub-GHz (433MHz, 915MHz)
  • Ideal para campos, florestas
  • Não é WiFi — não precisa internet
  • Módulo: ESP32 + LoRa (TTGO)

Imagem: Espressif Docs (Apache 2.0)

Infravermelho — Controle Remoto

  • IR — controle de TVs, split, etc.
  • Protocolos: NEC, Samsung, Sony...
  • TX: emissor IR (broadcast)
  • RX: receptor VS1838B (decode)
  • Biblioteca: IRremote.h
  • Automação residencial: comando universal

Imagem: RandomNerd Tutorials
