💤 O que é Light Sleep?

  • Modo de baixo consumo: ~0,4 mA
  • CPU pausada (não reinicia)
  • Wi-Fi desligado
  • Consome 200x menos que ligado normal (~80 mA)

    ⚖️ Light Sleep vs Deep Sleep

    Característica Light Sleep Deep Sleep
    Consumo ~0,4 mA ~0,02 mA
    CPU ⏸️ Pausada ❌ Desligada
    Reinicia? ✅ Não ❌ Sim
    Acorda com GPIO ou Timer GPIO 16 (RST)
    Uso de RAM ✅ Mantém ❌ Perde

    🔔 Por que usar Interrupt?

  • Economia inteligente: só acorda quando algo acontece
  • Sem timer desperdiçado: não precisa acordar só para verificar
  • Resposta imediata: age no instante do evento
  • Perfeito para sensores ambientais (luz, som, movimento)
    •

    Exemplo: detectar luz vs esperar 30 min para checar

    📷 LDR — Light Dependent Resistor

  • Resistor dependente de luz
  • Quanto mais luz → menor resistência
  • Quanto menos luz → maior resistência
  • Funciona como divisor de tensão
    •

    🔌 Circuito: LDR + Transistor → INTERRUPT

  • LDR positivo → Trimpot positivo
  • Trimpot negativo → GND
  • Wiper (centro) → base do transistor
  • Coletor → INTERRUPT pin (GPIO16/D0)
  • Emissor → GND

    🧊 O Transistor KSP2222A

  • Transistor NPN de uso geral
  • Funciona como interruptor eletrônico
  • Quando base recebe corrente → collector e emissor se conectam
  • Permite criar um sinal de interrupt no ESP8266
    •

    💡 O transistor "traduz" o sinal analógico do LDR em um sinal digital para o INTERRUPT

    📋 Pinagem do KSP2222A

  • Pino 1 (Emissor) → GND do circuito
  • Pino 2 (Base) → Wiper do Trimpot (sinal do LDR)
  • Pino 3 (Coletor) → GPIO16/D0 (INTERRUPT)
    •

    👋 "Olá, Detive Luz!" — Código

    
// Variáveis para controle
volatile bool luzDetectada = false;
unsigned long ultimoTempo = 0;

// Função de interrupção
voidICACHE_RAM_ATTRinterrompeLuz() {
  luzDetectada = true;
}

void setup() {
  Serial.begin(115200);
  Serial.println("Iniciando Light Sleep com Interrupt...");

  // Configura pino de interrupção (GPIO16/D0)
  pinMode(16, INPUT_PULLUP);
  attachInterrupt(digitalPinToInterrupt(16), interrompeLuz, CHANGE);

  Serial.println("Dormindo... (acorda com luz)");
}

void loop() {
  // Entra em Light Sleep até detectar luz
  if (!luzDetectada) {
    WiFi.mode(WIFI_OFF);
    wifi_fpm_set_sleep_type(LIGHT_SLEEP_T);
    wifi_fpm_open();
    wifi_fpm_set_wakeup_cb(callbackAcordou);
    wifi_fpm_do_sleep(0xFFFFFFF); // Dormir indefinidamente
    delay(10); // Necessário para ativar
  }

  // Ao acordar:
  Serial.println("☀️ LUZ DETECTADA! Acordei!");
  luzDetectada = false;

  // Executa tarefa (ex: enviar dados, imprimir)
  Serial.println("📡 Executando tarefa...");

  delay(5000); // Tempo para executar tarefa

  // Volta a dormir
  Serial.println("💤 Voltando a dormir...");
}
  • attachInterrupt() configura o pino GPIO16
  • wifi_fpm_do_sleep() entra em Light Sleep
  • Ao detectar luz, luzDetectada = true acorda o ESP
  • O código continua de onde parou (diferente do Deep Sleep!)

    🔔 Versão Simples: Print ao Detectar Luz

    
volatile bool luzDetectada = false;

voidICACHE_RAM_ATTRinterrompeLuz() {
  luzDetectada = true;
  Serial.println("☀️ LUZ DETECTADA!");
}

void setup() {
  Serial.begin(115200);
  pinMode(16, INPUT_PULLUP);
  attachInterrupt(digitalPinToInterrupt(16), interrompeLuz, CHANGE);
  Serial.println("💤 Dormindo... (aguarda luz)");
}

void loop() {
  if (!luzDetectada) {
    wifi_fpm_set_sleep_type(LIGHT_SLEEP_T);
    wifi_fpm_open();
    wifi_fpm_do_sleep(0xFFFFFFF);
    delay(10);
  }
  luzDetectada = false;
  
  Serial.println("📡 Fazendo algo útil...");
  delay(2000);
}
  • Interrupção imprime imediatamente ao detectar luz
  • Código continua após acorda, sem reiniciar

    ⚖️ Quando usar cada modo?

    Situação Recomendado
    Sensor de luz que só precisa reagir 🌙 Light Sleep
    Estação meteorológica (envio periódico) 💤 Deep Sleep
    Interruptor inteligente (reage a eventos) 🌙 Light Sleep
    Sensor de porta (dormir por meses) 💤 Deep Sleep
    Contador de visitantes 💤 Deep Sleep
