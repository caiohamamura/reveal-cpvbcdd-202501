Sensores Simples

Fáceis de usar, muito úteis no dia a dia

DS18B20 — Sensor de Temperatura Digital

  • 1-wire: apenas 1 pino de dados
  • Precisão: ±0.5°C (-10°C a +85°C)
  • Multiple sensores na mesma linha
  • Biblioteca: DallasTemperature

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

Boia — Sensor de Nível de Água

  • Interruptor magnético (reed switch)
  • Flutua com o nível da água
  • Saída digital: ON/OFF
  • Uso: reservas, bombas, lava-roupa

Fonte: MakerHero (CC BY-NC-SA)

Sensor Hall — Detecção de Campo Magnético

  • Detecta presença de ímã
  • Sem contato físico (desgaste zero)
  • Usado em: tacômetro, abertura de porta
  • Saída digital (HIGH/LOW)

Fonte: SparkFun (CC BY-SA 4.0)

Sensores de Distância

Medindo o mundo ao redor

HC-SR04 — Sensor Ultrassônico

  • Range: 2cm a 400cm
  • Precisão: 0.3cm
  • Usa som (40kHz), não luz
  • Funciona no escuro!

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

Sensor IR — Infravermelho

  • Sharp GP2Y0A21: range 10cm a 80cm
  • Saída analógica proporcional à distância
  • Mais preciso que ultrassônico em curta distância
  • Afetado pela luz ambiente

Fonte: Adafruit (CC BY-NC-SA 3.0)

VL53L0X — Sensor Laser de Tempo de Voo

  • Range: 50mm a 1000mm (1m)
  • Precisão: ±3%
  • Usa luz (laser 940nm), não som
  • I2C — fácil de conectar
  • Biblioteca: Adafruit_VL53L0X

Fonte: Adafruit (CC BY-NC-SA 3.0)

RCWL-0516 — Sensor Doppler (Micro-ondas)

  • Detecta movimento através de materiais
  • Usa micro-ondas (3.18GHz)
  • Penetra madeira, plástico, vidro
  • Range: 5-7 metros
  • Saída: LOW (standby) → HIGH (movimento)

Fonte: Circuits DIY (CC BY-SA)

Sensores Ambientais

Monitorando o ambiente

MQ-2 — Sensor de Gás e Fumaça

  • Detecta: GLP, Butano, Propano, Metano, Álcool, Hidrogênio
  • Saída: analógica (A0) + digital (D0 com threshold)
  • Sensibilidade ajustável via potenciômetro
  • Uso: detectors de incêndio, vazamento de gás

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

BME680 — Sensor Ambiental 4-em-1

  • Gás (VOC), Pressão, Umidade, Temperatura
  • Comunicação: I2C ou SPI
  • Ideal para: qualidade do ar interno (IAQ)
  • Biblioteca: Adafruit_BME680

Fonte: Adafruit (CC BY-NC-SA 3.0)

Sensor de Umidade do Solo

  • Mede resistência entre dois polos no solo
  • Solo seco = maior resistência
  • Solo úmido = menor resistência
  • Atenção: eletrólise — não deixar ligado 24h!

Fonte: MakerHero (CC BY-NC-SA)

Sensor de Chuva (Rain Module)

  • Placa com trilhas condutoras
  • Gotas de chuva curto-circuitam as trilhas
  • Saída: analógica + digital
  • Uso: automatização de toldos, janelas

Fonte: Usinainfo (CC BY-SA)

MPU-6050 — Acelerômetro + Giroscópio

  • 3 eixos: X, Y, Z aceleração
  • 3 eixos: X, Y, Z rotação (giroscópio)
  • I2C — usa biblioteca Adafruit_MPU6050
  • Uso: detecção de queda, movimento, orientação

Fonte: Adafruit (CC BY-NC-SA 3.0)

HC-SR501 — Sensor PIR (Infravermelho Passivo)

  • Detecta radiação IR de corpos quentes (pessoas, animais)
  • Não emite nada — passivo!
  • Range: ~7 metros, 120° cone
  • Ajuste: sensibilidade + tempo de delay

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

Atuadores — Motores e Movimento

Transformando sinais em movimento

Servo Motor — Controle de Posição

  • Rotação de 0° a 180° (micro servo)
  • Posição controlada por PWM (50Hz)
  • Biblioteca: Servo.h
  • Uso: portas, robôs, câmera, válvulas

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

Motor de Passo — Precisão de Posicionamento

  • Move em passos (steps) definidos
  • 28BYJ-48 + driver ULN2003
  • Precisão: 4096 passos / volta (ou 2048 half-step)
  • Uso: impressoras 3D, CNC, robôs

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

Solenóide — Movimento Linear Direto

  • Eletroímã que puxa/pusha um êmbolo
  • Precisa de transistor/diodo de proteção
  • Tipos: push (empurra) ou pull (puxa)
  • Uso: travas elétricas, disjuntores, bombas

Fonte: How2Electronics (CC BY-SA)

Comunicação IoT

Conectando seu projeto ao mundo

LoRa — Long Range, Baixo Consumo

  • Range: até 10km (line-of-sight)
  • Frequência: 433MHz / 868MHz / 915MHz
  • Baixo consumo (comparado a WiFi)
  • Uso: agricultura, sensores remotos, cidades inteligentes
  • Biblioteca: SPI + LoRa.h

Fonte: Adafruit (CC BY-NC-SA 3.0)

Infravermelho (IR) — Controle Remoto

  • Controle de TVs, DVDs, ar condicionado
  • Protocolos comuns: NEC, Sony, RC5
  • Biblioteca: IRremote.h
  • Transmissor + Receptor IR
  • Uso: automação residencial

Fonte: Adafruit (CC BY-NC-SA 3.0)

MQTT — Protocolo para IoT

  • MQTT = Message Queuing Telemetry Transport
  • Modelo publish/subscribe
  • Broker: servidor central (ex: Mosquitto)
  • Tópicos: "casa/sala/temperatura"
  • Ideal para: WiFi, rede celular
