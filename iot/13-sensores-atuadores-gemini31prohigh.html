Fase 1: Revisão Rápida

Vamos relembrar os componentes fundamentais já explorados.

Revisão: LDR + DHT22

  • LDR: Resistor dependente de luz (Analógico)
  • DHT22: Temperatura e Umidade (Digital)

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

Revisão: LCD I2C + Relé

  • LCD I2C: Display com apenas 2 pinos (SDA/SCL)
  • Relé: Chave eletromecânica (Atuador)

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

Revisão: Motor DC + MOSFET

  • Motor DC: Rotação contínua
  • MOSFET: Transistor para chaveamento de alta corrente
  • Controle de velocidade via PWM

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

Fase 2: Sensores Simples

Ampliando o leque de opções conhecidas.

Temperatura: DS18B20

  • Sensor de temperatura One-Wire
  • Ideal para medir líquidos (versão impermeável)
  • Múltiplos sensores em um único pino

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

Sensor Boia (Nível de Água)

  • Chave magnética (Reed Switch)
  • Circuito aberto/fechado conforme nível do líquido
  • Uso com pino digital (INPUT_PULLUP)
🛢️ 💧

Fase 3: Sensores de Distância

Diferentes tecnologias para medir proximidade.

Ultrassônico: HC-SR04

  • Utiliza som (eco) para medir distância
  • Faixa: 2cm a 400cm
  • Baixo custo, suscetível a ruído acústico/superfícies absorventes

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

Infravermelho e Laser (VL53L0X)

  • Infravermelho: Detecção simples de obstáculo (analógico/digital)
  • Laser (VL53L0X): Time-of-Flight (ToF)
  • Alta precisão e rapidez (I2C)
🔦 📏

Sensor Doppler (Radar)

  • Usa micro-ondas (Ex: RCWL-0516)
  • Detecta movimento através de paredes/objetos não metálicos
  • Ótimo para segurança oculta
📡 🌊

Fase 4: Sensores Ambientais

Monitoramento do ambiente físico.

Gás e Fumaça (Série MQ)

  • MQ-2 / MQ-135: Detectam gases inflamáveis, fumaça ou CO2
  • Possuem saída analógica (concentração) e digital (limite)
  • Exigem aquecimento interno
💨 ⚠️

Umidade do Solo e Chuva

  • Baseados em resistividade elétrica
  • Solo: Controle de irrigação agrícola
  • Chuva: Detecção de gotas numa placa exposta
🌱 🌧️

Acelerômetro e Giroscópio (MPU6050)

  • Aceleração em 3 eixos (X, Y, Z) e rotação espacial
  • Comunicação I2C
  • Usado em wearables, drones e estabilização

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

Sensor Hall e PIR

  • Sensor Hall: Detecta campos magnéticos (posição de portas, tacômetro)
  • Sensor PIR: Detecta radiação infravermelha de corpos (movimento humano)

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

Fase 5: Atuadores e Motores

Gerando ação física no mundo real.

Servo Motor

  • Controle preciso de ângulo (geralmente 0 a 180 graus)
  • Controle através de pulsos PWM
  • Aplicações: braços robóticos, trancas eletrônicas

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

Motor de Passo

  • Rotação controlada por passos exatos
  • Requer módulo driver (ex: ULN2003)
  • Excelente para CNCs, impressoras 3D e esteiras

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

Solenóide

  • Bobina eletromagnética que puxa/empurra um pino metálico
  • Movimento linear on/off (binário)
  • Requer uso de MOSFET/Relé pela alta corrente
🧲 🔓

Fase 6: Comunicação IoT

Opções para conectar todos esses componentes.

LoRa (Long Range)

  • Comunicação RF de longa distância (quilômetros)
  • Baixo consumo de energia e baixa taxa de dados
  • Ideal para IoT em áreas rurais ou Smart Cities

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

Infravermelho (Controle Remoto)

  • Transmissores (LED IR) e Receptores (Ex: TSOP4838)
  • Permite controlar TVs, ar-condicionados, etc.
  • Comunicação direcional com protocolo específico (ex: NEC)

Fonte: Random Nerd Tutorials (CC BY-NC-SA 4.0)

  • A escolha do componente depende sempre dos requisitos do projeto!
