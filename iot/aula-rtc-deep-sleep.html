    💤 Como o ESP8266 Dorme

    3 modos de sleep — do "cochilo" ao "coma"

    💤 Os 3 Modos de Sleep

    Modo Consumo CPU Wi-Fi Acordar com
    Modem Sleep ~15 mA ✅ Ligada ❌ Off Timer automático
    Light Sleep ~0,4 mA ❌ Pausada ❌ Off Timer ou GPIO
    Deep Sleep ~20 μA ❌ Off ❌ Off GPIO 16 (RST)

    💡 Deep Sleep desliga tudo. Só o RTC continua. Maior economia, mas reinicia do zero.

    💤 Deep Sleep em Detalhe

  • Tudo é desligado (CPU, Wi-Fi, RAM)
  • Só o RTC continua rodando
    1. Permite acordar após tempo
    2. Mantém memória RTC
  • Consome apenas 20 μA (0,02 mA)
  • Acorda como ligado do zero
  • Roda setup() novamente
    •
    Analogia 🔁: desligar o celular — o despertador continua

    ⚡ O Pino Mágico: GPIO 16 → RST

    • Conexão obrigatória: D0 → RST
    • É o D0 que "cutuca" o RST para acordar após tempo
    • Sem esse fio: o ESP dorme para sempre
    • Resistor 10kΩ entre RST e 3.3V (pull-up)

void loop() {
  // NUNCA executa após deep sleep!
}
  • 10e6 = 10 × 10⁶ μs = 10 segundos
  • Após deepSleep(), o ESP reinicia completamente
  • loop() nunca roda — tudo vai em setup()

    ⏱️ Matemática do Sleep

    ESP.deepSleep() recebe microssegundos

    Tempo Código Cálculo
    1 segundo 1e6 1 × 1.000.000
    10 segundos 10e6 10 × 1.000.000
    1 minuto 60e6 60 × 1.000.000
    5 minutos 300e6 300 × 1.000.000
    1 hora 3600e6 3600 × 1.000.000

    💡 Dica: use a fórmula segundos * 1e6 para nunca errar

    🔢 Calculadora de Autonomia

    Dados:

    • Bateria 18650: 3000 mAh
    • Acordado (Wi-Fi): 80 mA por 5 seg
    • Dormindo: 0,02 mA
    • Ciclo: a cada 30 minutos

    Cálculo por dia:

    • 48 ciclos/dia (24h ÷ 30min)
    • Acordado: 80 × 4min/60 = 5,3 mAh
    • Dormindo: 0,02 × 23,93h = 0,48 mAh
    • Total: ~5,8 mAh/dia

    Autonomia: 3000 ÷ 5,8 ≈ 517 dias! 🤯

    📊 Com vs Sem Deep Sleep

    Intervalo Sem Sleep Com Sleep Ganho
    Contínuo 37h — —

    📝 RTC Memory — O "Post-it" do ESP8266

  • 512 bytes que sobrevivem ao deep sleep
  • Acesso manual via ESP.rtcUserMemoryRead/Write
  • Rápido como RAM (sem desgaste de flash)
  • Não sobrevive a reset externo ou perda de energia

    #define SLEEP_CHUNK_US 3600000000ULL
#define TOTAL_CICLOS 2

struct Dados {
  int ciclos;

Dados dados;

void setup() {
  ESP.rtcUserMemoryRead(0, (uint32_t*)&dados, sizeof(dados));

  dados.ciclos++;

  if (dados.ciclos < TOTAL_CICLOS) {
    ESP.rtcUserMemoryWrite(0, (uint32_t*)&dados, sizeof(dados));
    ESP.deepSleep(SLEEP_CHUNK_US);
  } else {
    dados.ciclos = 0; // reset para próximo ciclo longo
    ESP.rtcUserMemoryWrite(0, (uint32_t*)&dados, sizeof(dados));

    // 👉 Aqui executa a tarefa após ~2h
  }
}

void onMqttMessage(char* topic, char* payload, AsyncMqttClientMessageProperties properties,
                   size_t len, size_t index, size_t total) {

  Serial.println("📥 Recebido: " + msg);

  // Converter mensagem para número
  contador = msg.toInt();
  recebeuValor = true;
}
