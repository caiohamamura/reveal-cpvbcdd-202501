    Etapa 1: Motivação e Critérios de Escolha

    Vocês terão cerca de 2 meses para o projeto final.

    Hoje o foco e ampliar repertório e decidir melhor entre alternativas.

    Como selecionar sensor ou atuador?

    • Faixa de operação: atende o cenário real?
    • Precisão: erro aceitável para a aplicação?
    • Consumo: vai rodar na bateria por quanto tempo?
    • Custo: cabe no orçamento da equipe?
    • Interface: digital, analógico, I2C, SPI, OneWire, UART?

    Etapa 2: Revisão Rápida

    Começamos do que vocês já dominam para conectar com os novos componentes.

    LDR + DHT22 (Revisão)

    • LDR converte luminosidade em variação resistiva.
    • DHT22 fornece temperatura e umidade com protocolo digital.
    • Bom par para monitoramento ambiental de baixo custo.

    Fonte: Random Nerd Tutorials

    LCD I2C + Relé (Revisão)

    • LCD I2C reduz fios para 2 linhas de dados (SDA/SCL).
    • Relé isola microcontrolador da carga externa.
    • Útil para ligar bomba, lâmpada ou ventilador de maior potência.

    Motor DC + MOSFET (Revisão)

    • MOSFET canal N funciona como chave eletrônica eficiente.
    • analogWrite() permite controlar velocidade por PWM.
    • Diodo flyback é obrigatório para proteger o circuito.

    Etapa 3: Sensores Simples

    Evoluindo do familiar para componentes de projeto final.

    DS18B20 - Temperatura OneWire

    • Protocolo OneWire: vários sensores no mesmo barramento.
    • Faixa ampla: -55 C a +125 C.
    • Precisão típica de +-0,5 C.
    • Versão encapsulada é ideal para ambiente úmido.

    Fonte: Random Nerd Tutorials

    Sensor de Nível por Boia (Float Switch)

    • Saída digital simples: nível atingido ou não.
    • Baixo custo e instalação direta em reservatórios.
    • Ótimo para proteção contra falta de água em bomba.

    Etapa 4: Sensores de Distância

    Ultrassom, laser e radar para cenários diferentes.

    HC-SR04 - Ultrassônico

    • Faixa típica: 2 cm a 4 m.
    • Bom custo-benefício para robótica e medição de nível.
    • Precisa de linha de visada e pode sofrer com superfícies inclinadas.

    Fonte: Random Nerd Tutorials

    VL53L0X - Laser Time-of-Flight

    • Mede tempo de voo do laser infravermelho.
    • Leitura estável para curtas distâncias com boa precisão.
    • I2C simplifica integração com outros sensores no barramento.

    Fonte: Adafruit

    Etapa 5: Sensores Ambientais

    Monitoramento de gás, solo, chuva, presença e movimento físico.

    MQ-2 - Gás e Fumaça

    • Detecta gases inflamáveis e fumaça.
    • Exige aquecimento inicial para estabilizar.
    • Saída analógica é útil para tendência; não para ppm absoluto sem calibração.

    Fonte: Random Nerd Tutorials

    Solo, Chuva, Presença e Campo Magnético

    • Capacitive Soil Moisture: umidade do solo sem corrosão.
    • Sensor de chuva: intensidade de água na superfície.
    • PIR: presença por variação de IR térmico.
    • Hall: campo magnético para contagem de voltas.

    Padrão de Código Não Bloqueante (IoT real)

    • Sem delay() longo travando o dispositivo.
    • Permite ler sensores e responder comandos ao mesmo tempo.
    • Facilita escalar para MQTT, Telegram ou dashboard web.

    Etapa 6: Atuadores e Movimento

    Comandos digitais virando ação física no mundo real.

    Servo SG90

    • Posicionamento angular preciso (0 a 180 graus).
    • Controle simples via biblioteca Servo.
    • Aplicações: fechadura, direcionamento de câmera, mini braços.

    Fonte: Adafruit

    Motor de Passo 28BYJ-48 + ULN2003

    • Movimento em passos discretos para melhor controle de posição.
    • Driver ULN2003 simplifica acionamento das bobinas.
    • Aplicações: dosador, persiana, automação de pequenos eixos.

    Fonte: Random Nerd Tutorials

    Relé vs MOSFET vs Solenóide

    Elemento Quando usar Vantagem Cuidado
