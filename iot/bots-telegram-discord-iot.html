Por que Telegram para IoT?

  • Bot API oficial e bem documentada
  • Biblioteca Arduino pronta (UniversalTelegramBot)
  • Comunicação direta ESP8266 ↔ Telegram (polling)
  • Comandos com prefixo /
  • Notificações push automáticas
  • Multi-plataforma (Android, iOS, Web)
  • Sem necessidade de servidores intermediários

📡 Arquitetura: ESP8266 ←→ Telegram (comunicação direta via polling)

    ☀️ LDR Proativo

    Projeto: Sensor de Luz Automático com Telegram

    🌡️ Histerese

    Evita liga/desliga constante (chattering)

    • Define dois limiares (liga e desliga)
    • Cria uma faixa de estabilidade
    • Depende do estado anterior (memória)

    Exemplo (Ar-condicionado):

    // Liga quando >= 25°C // Desliga quando <= 23°C if (temp >= 25 && estado == OFF) { ligar(); } if (temp <= 23 && estado == ON) { desligar(); }

    💡 Resultado: menos desgaste e comportamento estável
