Telegram vs Discord para IoT

Telegram ✅

  • Bot API oficial e bem documentada
  • Biblioteca Arduino pronta
  • Polling ou Webhook
  • Comandos com prefixo /
  • Notificações push automáticas
  • Multi-plataforma

Discord ⚡

  • Webhooks são simples e rápidos
  • Ideal para envio de dados
  • HTTP POST para canais
  • Sem biblioteca Arduino oficial
  • Formato JSON simples
  • Visual rico com embeds

⚠️ Segurança em Bots IoT

Boas práticas para proteger seus tokens e credenciais

Regras de Segurança

  • NUNCA exponha tokens no código versionado
  • Use secrets.h ou platformio.ini para tokens
  • Adicione arquivos sensíveis ao .gitignore
  • WiFiManager para credenciais WiFi (não hardcoded)
  • Valide todos os comandos recebidos
  • Use HTTPS para todas as comunicações
    •
    🔴 TENHO_TOKEN_TELEGRAM="123456:ABC-DEF..." // ❌ NUNCA FAÇA ISSO

    🤖 Bot Telegram

    Configurando e implementando um bot Telegram com ESP8266

    Criando Bot no Telegram

    1. Abra o Telegram e busque por @BotFather
    2. Envie /newbot e siga as instruções
    3. Escolha um nome e username para o bot
    4. Copie o token! Formato: 123456789:AABBcc...

    📱 Exemplo de conversa:

    /newbot → Nome: IoT Bot → Username: iot_esp8266_bot → Token: 6123456789:AAF...

    📶 WiFiManager

    Autoconfiguração de rede WiFi para dispositivos IoT

    O Problema

    ❌ Hardcoded WiFi credentials no código

    SSID: "MinhaRede"
    Password: "minhasenha123"

    Problemas:

    • Credenciais expostas no código
    • Precisa reprogramar para mudar rede
    • Não funciona em locais diferentes

    Solução: WiFiManager

  • Primeira vez: abre portal de configuração
  • AP modo: ESP8266 cria rede WiFi
  • Página web para selecionar rede
  • Salva credenciais na memória flash
  • Reconecta automaticamente
