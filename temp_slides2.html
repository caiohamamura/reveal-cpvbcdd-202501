--- SLIDE ---

4. Analisar no tempo

SELECT DATE_TRUNC('month', data) AS mes, SUM(valor) AS total FROM vendas GROUP BY mes ORDER BY mes;

Datas transformam registros soltos em serie historica: mes a mes, semana a semana, dia a dia.

Etapa 6: Integracao direta (Postgres)

O DuckDB possui uma extensao nativa para ler e escrever diretamente em bancos PostgreSQL.

INSTALL postgres; LOAD postgres; ATTACH 'dbname=meu_banco user=postgres host=localhost' AS pg_db (TYPE postgres); -- Agora podemos consultar tabelas do Postgres diretamente SELECT * FROM pg_db.public.vendas LIMIT 5;
--- SLIDE ---

Autenticacao no PostgreSQL

Na string de conexao

-- Senha inline ATTACH 'dbname=vendas user=postgres host=localhost password=minha_senha' AS pg (TYPE postgres);

Nunca versione senhas em arquivos de codigo.

Variaveis de ambiente

# Variáveis padrão PostgreSQL $env:PGHOST="localhost" $env:PGPORT="5432" $env:PGDATABASE="vendas" $env:PGUSER="postgres" $env:PGPASSWORD="minha_senha"

Com as variaveis definidas, basta ATTACH '' AS pg (TYPE postgres).

A extensao usa libpq internamente — todas as variaveis de ambiente padrao do PostgreSQL sao suportadas, incluindo o arquivo ~/.pgpass.
