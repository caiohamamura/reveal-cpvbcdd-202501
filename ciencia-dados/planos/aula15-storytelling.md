# Aula 15: Storytelling na Ciência de Dados

Esta aula vai servir como base para os alunos apresentarem e escreverem o caderno de análises do trabalho final. 

## Trabalho Final

No trabalho final os alunos vão receber um conjunto de dados reais obtidos por parceria com os professores e outros servidores. Baseado no que aprenderam no curso, eles devem realizar as análises pertinentes nos dados que serão apresentados a eles. 

As apresentações serão feitas em formato oral na penúltima semana do curso com a presença dos professores/servidores que cederam os dados. Os alunos irão efetivamente apresentar as análises realizadas para os "stakeholders". O caderno de análises também será entregue nesta semana aos professores/servidores.

## Considerações

Na aula anterior (aula14-mapa-analises.html) trabalhei diversos minicasos, demonstrando a maneira como podemos construir análises, reforçando a importância da pergunta como direcionador de toda a análise, assim como da elaboração de hipóteses a serem testadas e verificadas. Também mostrei na prática que tipos de análises escolheríamos para diversos tipos de dados e estudos, além de apresentar análise de séries temporais e PLN usando apenas machine learning tradicional sem deep learning. 

Nesta aula vamos trabalhar com storytelling. Vou apresentar como devemos pensar em storytelling na construção de uma análise, como devemos apresentar as análises, como devemos construir a narrativa de maneira sequencial e lógica, ainda que a análise realizada tenha sido feita de maneira não linear. Vou usar os minicasos da aula anterior para construir os storytelling e mostrar diferentes maneiras de apresentar as análises e estruturar os cadernos e apresentações.
