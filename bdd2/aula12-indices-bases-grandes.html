    Limpando linhas que não são válidas

  • Para limpar podemos usar diversas soluções:
  • Manualmente: abrir arquivo e apagar linhas (problema: trabalhoso e pode não ser possível para arquivos grandes)
  • Por scripts:
  • Python
  • Bash (linux)
  • Powershell (windows)
    •

    Quantas linhas vamos remover do início = 3

    Get-Content .\pam_21-23-var.csv -TotalCount 10 | Select-String . | Select-Object linenumber,line

    head pam_21-23-var.csv | nl

    Quantas linhas vamos remover do final = (40 - 7) = 33

    Get-Content .\pam_21-23-var.csv -Tail 40 | Select-String . | Select-Object linenumber,line

    tail -n 40 pam_21-23-var.csv | nl

    Removendo e jogando em um novo arquivo

  • # Configura diretório atual
  • [System.IO.Directory]::SetCurrentDirectory($PWD)
  • # Importa bibliotecas
  • $linq = [Linq.Enumerable];
  • $io = [System.IO.File];
  • # Lê as linhas do arquivo
  • $lines = $io::ReadLines("pam_21-23-var.csv")
  • # Conta linhas
  • $nlinhas = $linq::Count($lines);
  • # Pula 3 primeiras e pega as 33 últimas
  • $ok = $linq::Take($lines, $nlinhas - 33).Skip(3);
  • $io::WriteAllLines("pam_ok.csv", $ok)
    •

    head -n -33 pam_21-23-var.csv | tail -n +4 > pam_ok.csv

    Consultas lentas e índices

    Índices em bancos de dados relacionais

  • Funcionam de forma análoga aos índices de livros
  • SGBD-R armazenam em um formato específico dependendo do tipo de índice
  • Os índices são baseados em uma ou mais colunas
    •
  • Vantagens
  • Acelera consultas (muito)
    •
  • Desvantagens
  • Ocupa espaço
  • Deixa inserções, atualizações e remoções mais lentas
    •

    Quando e como usar índice?

    Analisando necessidade de índice

  • Caso queira analisar se uma consulta precisa de índice use a função:
  • EXPLAIN ANALYZE
  • Para descobrir o que melhorar procure as partes com maior incremento de custo (cost) e por palavras-chave como:
  • Seq Scan
  • Nested loop
    •

    Como os índices funcionam?

  • O índice usado normalmente por SGBD é o B-tree
  • É semelhante a uma árvore binária, mas tem algumas peculiaridades que a deixam mais adequadas como:
  • Balanceamento automático
  • Otimizações para ajuste ao atualizar, inserir ou apagar
