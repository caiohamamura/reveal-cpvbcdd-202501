    💤 Como o ESP8266 Dorme

    3 modos de sleep — do "cochilo" ao "coma"

    💤 Os 3 Modos de Sleep

    Modo Consumo CPU Wi-Fi Acordar com
    Modem Sleep ~15 mA ✅ Ligada ❌ Off Timer automático
    Light Sleep ~0,4 mA ❌ Pausada ❌ Off Timer ou GPIO
    Deep Sleep ~20 μA ❌ Off ❌ Off GPIO 16 (RST)

    💡 Deep Sleep desliga tudo. Só o RTC continua. Maior economia, mas reinicia do zero.

    💤 Deep Sleep em Detalhe

  • Tudo é desligado (CPU, Wi-Fi, RAM)
  • Só o RTC continua rodando
  • Consome apenas 20 μA (0,02 mA)
  • Acorda como ligado do zero
  • Roda setup() novamente
    •
    📐 Analogia: desligar o celular — o despertador continua, mas tudo mais reinicia

    ⚡ O Pino Mágico: GPIO 16 → RST

  • Conexão obrigatória: D0 (GPIO 16) → RST
  • É o RTC que "cutuca" o RST para acordar
  • Sem esse fio: o ESP dorme para sempre
  • Resistor 10kΩ entre RST e 3.3V (pull-up)

    📝 RTC Memory — O "Post-it" do ESP8266

  • 512 bytes que sobrevivem ao deep sleep
  • Declarado com RTC_DATA_ATTR
  • Rápido como RAM (sem desgaste de flash)
  • Não sobrevive a desconexão de energia
    •  
    
RTC_DATA_ATTR int contador = 0;            // Persiste ao acordar!
RTC_DATA_ATTR float ultimaTemperatura = 0;  // Persiste ao acordar!
RTC_DATA_ATTR bool primeiraVez = true;      // Persiste ao acordar!

    🧠 O Que Sobrevive?

    Tipo Deep Sleep? Power-off? Velocidade Limite
    RAM normal ❌ Perde ❌ Perde Rápida ~80 KB
    RTC memory ✅ Mantém ❌ Perde Rápida 512 bytes
    EEPROM ✅ Mantém ✅ Mantém Média 4 KB
    SPIFFS ✅ Mantém ✅ Mantém Lenta ~3 MB

    💡 Para deep sleep, RTC memory é ideal: rápida e persiste. Para dados críticos, use SPIFFS.

    ⏰ NTP Inteligente — A Economia

  • 1ª vez: conecta Wi-Fi → sincroniza NTP → salva epoch
  • Depois: calcula hora via millis() - millisBase + epochBase
  • Economia: Wi-Fi ligado só no primeiro ciclo!
    •

    ⚡ Com Wi-Fi: 5 seg × 80mA = 400 mA·s. Sem Wi-Fi: 5ms × 20mA = 0,1 mA·s → 4.000x menos por ciclo!

    ⏰ Código: NTP + RTC Memory

    
RTC_DATA_ATTR bool ntpOk = false;
RTC_DATA_ATTR time_t epochBase = 0;
RTC_DATA_ATTR unsigned long millisBase = 0;

void setup() {
  Serial.begin(115200);

  if (!ntpOk) {
    // Primeira vez: conecta e sincroniza
    WiFi.begin(ssid, senha);
    configTime(0, 0, "pool.ntp.org");
    while (time(nullptr) < 1000) delay(100);
    epochBase = time(nullptr);
    millisBase = millis();
    ntpOk = true;
    WiFi.disconnect(true);  // Desliga Wi-Fi!
  }

  // Calcula hora sem Wi-Fi
  time_t agora = epochBase + (millis() - millisBase) / 1000;
  Serial.printf("Hora: %s", ctime(&agora));

  ESP.deepSleep(300e6);  // 5 minutos
}

void loop() {}

RTC_DATA_ATTR unsigned long millisBase = 0;

void setup() {
  Serial.begin(115200);

  if (!ntpOk) {
    WiFi.begin(ssid, senha);
    configTime(0, 0, "pool.ntp.org");       // ⬅ sincroniza
    while (time(nullptr) < 1000) delay(100); // ⬅ espera NTP
    epochBase = time(nullptr);
    millisBase = millis();
    ntpOk = true;
    WiFi.disconnect(true);

  time_t agora = epochBase +                // ⬅ calcula sem Wi-Fi
    (millis() - millisBase) / 1000;          // ⬅ diferença de tempo
  Serial.printf("Hora: %s", ctime(&agora));
  ESP.deepSleep(300e6);
}
